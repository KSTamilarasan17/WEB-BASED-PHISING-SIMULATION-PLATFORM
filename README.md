# WEB-BASED-PHISING-SIMULATION-PLATFORM

A Web-Based Phishing Simulation Platform designed to educate users and organizations about phishing attacks by simulating real-world phishing scenarios in a safe and controlled environment.
This project helps users identify phishing techniques, understand attacker behavior, and improve cybersecurity awareness.
